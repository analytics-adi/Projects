# Swiggy Sales-Customers Analysis Dashboard

#Project Overview
This project analyzes Swiggy sales data using Power BI to uncover business insights, customer behavior, and sales performance trends.

# Tools Used
- Power BI
- SQL
- Excel

 # Key KPIs
- Total Revenue

- Average Order Value
- Customer Count
- Sales Growth

- Tracked sales trends over time
